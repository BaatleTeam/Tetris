# Tetris
Project "Tetris" is a classic game with GUI written in **C++** with **SFML** graphics library.

## Compiling

### Dependencies
| Dependency | Version |
|------------|---------|
| GCC        | 7.3+    |
| CMake      | 3.3+    |
| SFML       | 2.5+    |


### Build
Build while in root of project directory:
```
cmake .
make
```
Run compiled application:
```
./Build/Tetris.exe
```
